# Pre-Send Tone Checklist

Run this before sending any template. Takes 60 seconds. If any box is unchecked, do not send.

---

## Universal Checks (all templates)

- [ ] **Hook present** — sentence 1 explains why THIS person specifically, not a generic opener
- [ ] **Proof present** — at least one specific number (minutes, rupiah, percent, or count)
- [ ] **Ask present** — one clear, low-friction next step in the last sentence
- [ ] **No emojis** — zero emojis anywhere in the message
- [ ] **No fabricated context** — no "as we discussed", "as I mentioned", "you expressed interest"
- [ ] **Placeholders filled** — no `[brackets]` left in the message
- [ ] **App link correct** — any link used is https://app.larinova.com, not an old domain
- [ ] **Tagline correct** — if tagline appears, it reads "AI Medical Scribe for Doctors", NOT "Clinic Management & AI Platform"
- [ ] **No paid conversion ask** — message does not ask them to pay, subscribe, or upgrade (free-trial phase only)
- [ ] **Send time** — it is 07:00–08:00 or 19:00–20:00 local time (not 09:00–12:00 clinical hours)

---

## Bahasa Indonesia Checks

- [ ] **Self-reference** — uses "saya" throughout, never "aku" in cold outreach
- [ ] **Doctor address** — uses "Dokter", "Bapak Dokter", or "Ibu Dokter"; never "kamu" or "Anda" for the doctor
- [ ] **Company reference** — "Larinova" or "kami" (not "kita" which implies shared group membership)
- [ ] **Free offer wording** — "gratis 1 bulan, tanpa kartu kredit" (not "trial" or "masa coba")
- [ ] **Localization Reviewer approved** — stamp/sign-off from Localization Reviewer agent on file

---

## English (India) Checks

- [ ] **Tamil tagline present** — WA templates include the Tamil tagline at the end where specified
- [ ] **No "I hope this finds you well"** — banned opener
- [ ] **Respectful address** — "Dr. [Surname]" on first touch; first name only after they reply
- [ ] **ABDM/Sarvam mention** — one of these is included if the template targets a clinical buyer (not ambassador)

---

## LinkedIn-Specific Checks

- [ ] **Connection request ≤30 words** — count before sending
- [ ] **Post-connection message ≤60 words** — count before sending
- [ ] **No automation signature** — message reads as personally written, not auto-generated
- [ ] **Sent from Gabriel's personal LinkedIn** — not a company page

---

## Email-Specific Checks

- [ ] **Subject line ≤50 characters** — count characters including spaces
- [ ] **Body ≤120 words** — count before sending
- [ ] **Sender address** — hello@larinova.com only
- [ ] **No "noreply"** — sending from hello@ not noreply@ or onboarding@resend.dev
- [ ] **Signature includes WhatsApp offer** — "Kalau ada pertanyaan, langsung balas atau chat WhatsApp"

---

## After Sending

- [ ] CRM logged: date, channel, template ID used, doctor name
- [ ] Bump reminder set: Day +2 in calendar if WA, Day +3 if email
- [ ] A/B log updated: add 1 tally to template variant used this week
